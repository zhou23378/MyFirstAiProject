# 需求规划 — 对标客满满功能全景

> 客满满（美团旗下美业 SaaS）是行业标杆，覆盖单店到连锁的完整经营场景。
> 本文档逐功能对标，明确已实现、未实现、差距优先级。每次功能迭代后更新对应条目。
